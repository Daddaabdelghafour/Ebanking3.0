# Angular Frontend

Angular frontend application for E-Bank 3.0.

**Responsible**: Khalid

## Features

- GraphQL client integration
- User authentication
- Account management UI
- Payment processing UI
- Notification center

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Setup

```bash
npm install
```

### Run Development Server

```bash
npm start
```

The application will be available at `http://localhost:4200`

### Build for Production

```bash
npm run build
```

## Configuration

Update `src/main.ts` to configure the GraphQL endpoint URL based on environment.
